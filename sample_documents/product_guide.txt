제품 가이드 - SmartWidget Pro

1. 제품 개요
SmartWidget Pro는 AI 기반 데이터 분석 플랫폼입니다.
실시간 데이터 처리와 시각화를 지원하며, 비개발자도 쉽게 사용할 수 있습니다.

2. 주요 기능
- 실시간 대시보드: 데이터를 실시간으로 모니터링할 수 있습니다.
- AI 분석: 머신러닝 모델을 활용한 자동 인사이트 추출
- 리포트 생성: 원클릭으로 PDF/Excel 리포트 생성 가능
- 알림 설정: 임계치 기반 자동 알림 (이메일, 슬랙, 문자)
- API 연동: REST API를 통한 외부 시스템 연동

3. 요금제
- Free: 월 1,000건 데이터, 대시보드 3개, 무료
- Standard: 월 100,000건 데이터, 대시보드 무제한, 월 49,000원
- Professional: 월 1,000,000건 데이터, AI 분석 포함, 월 149,000원
- Enterprise: 무제한 데이터, 전용 서버, 별도 협의

4. 기술 사양
- 지원 데이터 소스: MySQL, PostgreSQL, MongoDB, CSV, REST API
- 응답 시간: 평균 200ms 이내
- 가용성: 99.9% SLA 보장
- 데이터 보안: AES-256 암호화, SOC2 인증

5. 자주 묻는 질문
Q: 무료 체험이 가능한가요?
A: 네, 14일간 Professional 플랜을 무료로 체험할 수 있습니다.

Q: 데이터 이전이 가능한가요?
A: CSV, JSON 형식으로 데이터 가져오기/내보내기가 가능합니다.

Q: 온프레미스 설치가 가능한가요?
A: Enterprise 플랜에서 온프레미스 설치를 지원합니다.
