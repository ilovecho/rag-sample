## 다음 프로젝트 아이디어

### 초급 프로젝트
```
1. 내 노트 검색기
   - 자신의 메모/노트 파일로 RAG 구축
   - "지난달에 배운 파이썬 팁이 뭐였지?" 같은 검색

2. FAQ 챗봇
   - 자주 묻는 질문 목록으로 자동 답변 봇
   - 고객 지원, 사내 헬프데스크에 활용

3. 독서 노트 도우미
   - 읽은 책의 핵심 내용을 저장하고 검색
   - "동기부여에 대해 어떤 책에서 뭐라고 했지?"
```

### 중급 프로젝트
```
4. 기술 문서 검색 시스템
   - API 문서, 기술 블로그를 RAG로 검색
   - Streamlit으로 웹 UI 구현

5. 법률/규정 QA 시스템
   - 사내 규정, 법률 문서를 기반으로 QA
   - 메타데이터로 조항/섹션별 필터링

6. 다국어 RAG
   - 영문 문서를 한국어로 질문하여 답변 받기
   - 번역 체인과 RAG 결합
```

### 고급 프로젝트
```
7. 멀티모달 RAG
   - 이미지, 표, 차트가 포함된 문서 처리
   - GPT-4V와 결합하여 이미지 기반 QA

8. Agent + RAG
   - RAG를 도구로 사용하는 AI 에이전트
   - 필요할 때만 문서 검색, 나머지는 자체 판단

9. 실시간 RAG
   - 웹 크롤링으로 최신 정보 자동 수집
   - 벡터DB 자동 업데이트 파이프라인
```

## 핵심 라이브러리 정리

| 라이브러리 | 역할 | 설치 |
|-----------|------|------|
| `langchain` | RAG 프레임워크 | `pip install langchain` |
| `langchain-openai` | OpenAI 연동 | `pip install langchain-openai` |
| `langchain-community` | 커뮤니티 통합 | `pip install langchain-community` |
| `chromadb` | 벡터 DB | `pip install chromadb` |
| `faiss-cpu` | 벡터 DB (빠름) | `pip install faiss-cpu` |
| `pypdf` | PDF 처리 | `pip install pypdf` |
| `tiktoken` | 토큰 카운팅 | `pip install tiktoken` |
| `streamlit` | 웹 UI | `pip install streamlit` |
| `gradio` | 웹 UI (간편) | `pip install gradio` |

## 학습 팁

```
1. 코드를 복사하지 말고 직접 타이핑하세요
   → 손으로 치면서 이해가 깊어집니다

2. 파라미터를 바꿔가며 실험하세요
   → chunk_size, k, temperature 등을 바꿔보면 동작 원리를 체감

3. 자신의 문서로 시도하세요
   → 샘플 문서 대신 실제 관심 있는 문서로 테스트

4. 에러를 두려워하지 마세요
   → 에러 메시지는 최고의 학습 자료입니다

5. 중간 결과를 항상 출력해보세요
   → print()로 각 단계의 결과를 확인하면 흐름이 보입니다
```
